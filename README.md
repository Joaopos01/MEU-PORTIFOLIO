# Portfólio | João Pedro

Este é meu portfólio pessoal, desenvolvido com **HTML, CSS e JavaScript**.

O objetivo do projeto é apresentar minha trajetória, habilidades, projetos e formas de contato de maneira simples, organizada e responsiva.

## 📌 Sobre o projeto

Este portfólio foi criado para reunir minhas principais informações como estudante de Engenharia de Software e desenvolvedor em formação.

Nele, apresento um pouco sobre mim, minhas habilidades técnicas, meus projetos e meus canais de contato profissional.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub

## 🖥️ Como visualizar o projeto

Você pode clonar este repositório e abrir o arquivo `index.html` no navegador.

```bash
git clone https://github.com/Joaopos01/MEU-PORTFOLIO.git
```

Depois, entre na pasta do projeto:

```bash
cd MEU-PORTFOLIO
```

Abra o arquivo `index.html` no navegador.
